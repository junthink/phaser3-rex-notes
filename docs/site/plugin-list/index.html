    Plugin list

    List of my plugins¶

    Version of phaser3

    Please upgrade your phaser3 to version >= 3.55.2.

    1. Achievements: Achievements in a csv table.
    2. Anchor: Set size and position based on visible window.
    3. Arcade-TCRP/Player: Run commands on step of Arcade.
    4. Arcade-TCRP/Recorder: Store commands with step of Arcade.
    5. AwaitLoader: Await custom task in preload stage.
    6. Away time: Get time from previous closing application to now.
    7. BBCodeText: Drawing text with BBCode protocol.
    8. Board: Core object of Board system.
    9. Board/Field of view: Visible testing, to find field of view.
    10. Board/Hexagon map: Create tile positions in hexagon/triangle/parallelogram geometry in hexagon grid.
    11. Board/Match: Get matched chess.
    12. Board/Mini board: Chess Container, to rotate/mirror/drag chess together.
    13. Board/Monopoly: Move through path tiles.
    14. Board/Move To: Move chess towards target position with a steady speed.
    15. Board/Path finder: Find moveable area or moving path.
    16. Board/Shape: Grid (polygon) shape object.
    17. Bracket parser: A lite-weight bracket parser.
    18. Buff data: Data manager with buffs.
    19. Build arcade object: Create arcade body, and inject arcade object methods.
    20. Bullet: Move game object toward current angle of game object, with a constant speed.
    21. Button: Fires 'click' event when touch releasd after pressed.
    22. Canvas: Drawing on canvas.
    23. Canvas/Circular progress: Circular progress bar on canvas.
    24. Canvas/Round rectangle: Round rectangle on canvas.
    25. Canvas image data: Get image data from texture, or text object.
    26. Canvas frame manager: Generate bitmapfont from text game object, or bbcode text game object.
    27. Circle mask image: Load a texture, then apply a circle mask.
    28. Click outside: Fires 'clickoutside' event when pointer-down or pointer-up outside of game object.
    29. Clock: A clock to count elapsed time.
    30. Character cache: Generate bitmapfont from text game object, or bbcode text game object.
    31. Color/TintRGB: Attach tintR, tintG, tintB, and tintGray properties to a game object.
    32. Conditions table: Check conditions to find passed tests listed in a csv table.
    33. ContainerLite: Control the position and angle of children game objects.
    34. Containerlite-perspective: Snapshot children of containerlite, to a perspective render texture.
    35. CSV-scenario: Run script in csv format.
    36. CSV-to-hash-table: Hash table indexed by (col-key, row-key) from csv string.
    37. Cursor at bound: Map position pf cursor to cursor key state.
    38. Curve/Spiral: Spiral curve.
    39. Drag: Drag game object.
    40. Drag-rotate: Get dragging angle around a specific point.
    41. Dynamic text: Control position, angle of each character drawn on a canvas.
    42. Ease-data: Easing data value of game object's data-manager.
    43. Ease-move: Ease-move game object.
    44. Eight direction: Move game object by cursor keys, with a constant speed.
    45. Event promise: Return a promise of an event.
    46. Expression parser: Parse expression string into function.
    47. Fade-out-destroy: Fade out game object then destroy it.
    48. Fade-volume: Fade-in/fade-out volume of sound.
    49. File chooser: Create a transparent file chooser button.
    50. Firebase/broadcast: Broadcast real-time messages, using firebase-database.
    51. Firebase/files: Save JSON data, using firebase-firestore.
    52. Firebase/ID-alias: Map an unique ID to another unique ID, using firebase-firestore.
    53. Firebase/Item-table: 1d/2d/3d table, using firebase-database.
    54. Firebase/leaderboard: Descending sort scores, using firebase-firestore.
    55. Firebase/messages: Store messages in firebase-firestore.
    56. Firebase/online-user-list: Online user list, using firebase-database.
    57. Firebase/single-room: Chat room, using firebase-database.
    58. Flash: Flashing (set invisible then visible) game object.
    59. Flip: Flipping game object to another face by scaling width/height.
    60. FSM: Finite state machine.
    61. Fuzzy: A wrap of fuzzy logic.
    62. Gashapon: Pick random item from box.
    63. Geom/Hexagon: Hexagon geometry object.
    64. Geom/Rhombus: Rhombus shape and methods.
    65. Gesture/Pan: Get pan events of a game object.
    66. Gesture/Pinch: Get scale factor from 2 dragging touch pointers.
    67. Gesture/Press: Get press events of a game object.
    68. Gesture/Rotate: Get spin angle from 2 dragging touch pointers.
    69. Gesture/Swipe: Get swipe events of a game object.
    70. Gesture/Tap: Get tap/multi-taps events of a game object.
    71. Grid align: Align objects on quadrilateral or hexagon grid.
    72. Grid table: Viewer of grid table, to manipulate game object of each visible cell.
    73. Hidden input text: An invisible Input DOM element to receive character input and display on text game object.
    74. Image URI loader: Load image by uri in preload stage.
    75. Input text: Input DOM element.
    76. Interception: Predict the intersection position of two game objects with constant moving speed.
    77. In touching: Fires 'intouch' event every tick when pressing on a game object.
    78. Life time: Destroy game object when time-out.
    79. Line: Draw a line with start/end/body textures.
    80. Live2d: Display live2d model.
    81. LocalForage/files: Save JSON data, using localforage.
    82. LocalStorage/data: Sync data from data manager to local-storage.
    83. LZ-string: Compress string using LZ-based compression algorithm.
    84. Modal promise: Modal behavior wrapped into promise.
    85. Mouse-wheel to up/down: Map mouse-wheeling to (up/down) cursor key state.
    86. Mouse-wheel scroller: Emit scroll event when mouse-wheeling.
    87. Move to: Move game object towards target position with a steady speed.
    88. Nine patch: Stretchable image.
    89. Nine patch2: Stretchable image. Has better performance than nine-patch.
    90. Particles-along-bounds: Emit particles along bounds of game object.
    91. Particles/Bitmap-zone: Particles' emitter zone from canvas bitmap of text/canvas game object.
    92. Path follower: Set position of game object on a path.
    93. Perlin: Perlin2/Perlin3 noise and simplex2/simplex3 noise.
    94. Perspective/Card: A container with two perspective-images.
    95. Perspective/Carousel: A container with cards.
    96. Perspective/Image: Image with perspective rotation.
    97. Perspective/ImageCarousel: Display list of images.
    98. Perspective/Render texture: Render texture with perspective rotation.
    99. Perspective/Sprite: Play animation on perspective-image..
    100. polar-coordinate: Attach polarOX, polarOY, polarRotation, polarAngle, and polarRadius properties to a game object.
    101. Pop up: Scale up game object.
    102. Quad/Image: Image with 4 or 9 vertex control points.
    103. Quad/Render texture: Render texture with 4 or 9 vertex control points.
    104. Quest: Question manager.
    105. Raycaster: Raycaster between obstacles.
    106. Random place: Place objects randomly inside an area without overlapping.
    107. Real-time timers: Start and counting timer by real-time timestamp.
    108. Restorable data: Restorable data manager.
    109. RotateTo: Rotate game object towards target position with a steady speed.
    110. Run-commands: Run commands in array.
    111. Scale-down-destroy: Scale down game object then destroy it.
    112. Scale outer: Scroll and zoom camera to make default game window fit the display area, in RESIZE scale mode.
    113. Script-tag loader: Load script tag in preload stage.
    114. Scroller: Drag content. Slow down when dragging released, pull back when out of bounds.
    115. Sequence: Run sequence commands in array.
    116. Shader/barrel: Barrel post processing filter.
    117. Shader/color-replace: Replace color post processing filter.
    118. Shader/cross-stitching: Cross-stitching post processing filter.
    119. Shader/dissolve: Dissolve transition post processing filter.
    120. Shader/drop-shadow: Drop-shadow post processing filter.
    121. Shader/fish-eye: Fish-eye post processing filter.
    122. Shader/glow-filter: Glow post processing filter.
    123. Shader/glow-filter: Glow post processing filter, ported from pixi.
    124. Shader/gray-scale: Gray scale post processing filter.
    125. Shader/hsl-adjust: Adjust color in HSL domain, post processing filter.
    126. Shader/inverse: Inverse color post processing filter.
    127. Shader/kawaseblur: Kawase-blur post processing filter.
    128. Shader/outline: Outline post processing filter, ported from pixi.
    129. Shader/pixelation: Pixelation post processing filter.
    130. Shader/shockwave: Shockwave post processing filter.
    131. Shader/split: Split image into 4 parts.
    132. Shader/swirl: Swirl post processing filter.
    133. Shader/toonify: Draw outlines and quantize color in HSV domain, post processing filter.
    134. Shake-position: Shake position of game object.
    135. Shape/Circular progress: Circular progress bar shape.
    136. Shape/Cover: Rectangle shape covered full window, and block all touch events.
    137. Shape/Custom progress: Custom progress bar shape.
    138. Shape/Custom shapes: Custom shapes on shape.
    139. Shape/Full window rectangle: Rectangle shape covered full window.
    140. Shape/Round-Rectangle: Round rectangle shape.
    141. Shape/Spinner: Loading animations on shape.
    142. Shatter/Image: Shatter image to triangle faces.
    143. Shatter/Render texture: Shatter render texture to triangle faces.
    144. Ship: Move game object as a space ship by cursor keys.
    145. Slider: Drag thumb on a slider bar.
    146. State manager: Another version of Finite state machine.
    147. TagText: Displays text with multi-color, font face, or font size with tags.
    148. TCRP/Player: Run commands on time.
    149. TCRP/Recorder: Store commands with time.
    150. Text/edit: Create an input text object above a text object to edit string content.
    151. Text/typing: Typing text on text object.
    152. Text/page: Display text page by page on text object.
    153. Text player: Typing characters on dynamic text, waiting click or key enter, play sound effect or backgroun music.
    154. Touch event stop: Stop touch events propagation.
    155. Transition image: Transit texture to another one.
    156. UI/badge label: A container with badges above a main item.
    157. UI/buttons: A container with a group of buttons.
    158. UI/chart: Draw chart on canvas.
    159. UI/dialog: A container with a title, content, buttons and backgrounds.
    160. UI/fixwidthbuttons: A container with a group of fix-width buttons.
    161. UI/fixwidthsizer: Layout children game objects into lines.
    162. UI/grid buttons: A container with a group of buttons in grids.
    163. UI/grid sizer: Layout children game objects in grids.
    164. UI/gird table: A container with a grid table, slider, and scroller.
    165. UI/holy grail: Layout elements in Holy grail style.
    166. UI/Knob: A knob button based on circular progress.
    167. UI/label: A game object container with an icon, text, and background.
    168. UI/menu: A container with buttons and sub-menu.
    169. UI/number bar: A container with an icon, slider, text, and background.
    170. UI/overlap sizer: Layout children game objects overlapped.
    171. UI/pages: A container with pages, only current page is visible.
    172. UI/scroll-able panel: A container with a panel, slider, and scroller.
    173. UI/scroll bar: A container with slider, two buttons, and background.
    174. UI/sizer: Layout children game objects.
    175. UI/slider: A container with a track, indicator, thumb and background.
    176. UI/tabs: A container with 4 groups of buttons around a center panel.
    177. UI/text area: A container with a text, slider, and scroller.
    178. UI/text box: A container with an icon, (typing and paging) text, and background.
    179. UI/toast: Show text message for a short while.
    180. Unique item list: List of unique items. Support array and set methods.
    181. Virtual joystick: Simulate cursor keys according to touch events.
    182. Wait events: Wait fired events or callbacks.
    183. Webfont-loader: Load web font by google webfont loader in preload stage.
    184. XOR: Encrypt or decrypt string by XOR algorithm.
    185. YML-Achievements: Achievements in a YAML table.
    186. YML-Conditions table: Check conditions to find passed tests listed in a YAML table.
    187. Youtube player: Play youtube video on iframe.
